================================================================================
  FLUJO CORRECTO DE prepare_data()
================================================================================

ENTRADA:
    prepare_data(df: DataFrame, target: str, problem_type: str = 'auto')
    
    Donde:
    - df: DataFrame CON la columna target incluida
    - target: nombre de la columna target (string)
    - problem_type: 'auto', 'binary', 'multiclass', 'regression'

SALIDA:
    return X_raw, y, preprocessor, meta
    
    Donde:
    - X_raw: DataFrame con features (sin target)
    - y: Array con target codificado
    - preprocessor: ColumnTransformer para procesar nuevos datos
    - meta: Dict con metadatos


COMPARACIÓN - ANTES vs DESPUÉS
────────────────────────────────

❌ ANTES (incorrecto):
    # Pasar X e y por separado - INCORRECTO
    preprocessor, X_processed, meta = prepare_data(X, y, problem_type='auto')
    # 3 variables para 4 valores → ValueError

✅ DESPUÉS (correcto):
    # Crear DataFrame completo
    data_with_target = X.copy()
    data_with_target['aprobado'] = y
    
    # Desempacar 4 valores correctamente
    X_raw, y_processed, preprocessor, meta = prepare_data(

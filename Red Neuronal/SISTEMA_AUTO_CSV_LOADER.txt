1. ✓ CREADO: csv_auto_loader.py (267 líneas)
   ───────────────────────────────────────────────────────────────────────────
   
   Módulo completo de detección automática con:
   
   • get_available_datasets()
     - Escanea dinámicamente la carpeta 'datos/'
     - Retorna diccionario {nombre: Path}
     - Valida cada CSV antes de incluirlo
     - Ordenado alfabéticamente
   
   • _validate_csv(csv_path)
     - Valida existencia y legibilidad del archivo
     - Verifica mínimo de 2 columnas y 1 fila de datos
     - Valida tamaño máximo (100MB)
     - Detecta valores NaN excesivos
     - Retorna Tuple[bool, str] con resultado
   
   • get_dataset_info(dataset_name)
     - Retorna información detallada del dataset
     - Incluye: filas, columnas, tamaño, tipos, memoria
     - Útil para debugging y auditoría
   
   • print_available_datasets()
     - Visualización formateada de todos los datasets
     - Muestra detalles de estructura
     - Facilita monitoreo de qué datasets están disponibles
   
   • watch_for_new_csvs(callback, poll_interval)
     - Monitor continuo de cambios en datos/
     - Ejecuta callback cuando detecta cambios
     - Opcional: para aplicaciones 24/7
     - Requiere: pip install watchdog
   
   • create_test_csv(filename, rows)
     - Utilidad para crear CSV de prueba
     - Para testing y demostración

──────────────────────────
1. Usuario agrega archivo.csv a carpeta 'datos/'
2. Sistema detecta automáticamente en siguiente ejecución
3. Aparece automáticamente en menú de selección
4. Sin cambios en código
5. RESULTADO: Automático, robusto, inmediato


PASO A PASO - QUÉ OCURRE:
─────────────────────────

1. Usuario inicia main.py
   ↓
2. Se llama a _get_demo_csv_by_fuente()
   ↓
3. Función llama a get_available_datasets()
   ↓
4. csv_auto_loader.py:
   a) Abre carpeta 'datos/'
   b) Busca todos los archivos *.csv
   c) Para cada CSV:
      - Valida estructura (columnas, filas, tamaño)
      - Si es válido → lo incluye
      - Si es inválido → lo salta (con warning)
   d) Retorna diccionario ordenado alfabéticamente
   ↓
5. main.py genera menú dinámico con los datasets encontrados
   ↓
6. Usuario ve lista automática de opciones

VALIDACIONES AUTOMÁTICAS
═══════════════════════════════════════════════════════════════════════════════

Cada CSV se valida automáticamente contra:

✓ Existencia y legibilidad del archivo
✓ Formato válido de CSV (parseable con pandas)
✓ Mínimo de 2 columnas
✓ Mínimo de 1 fila de datos (aparte de encabezado)
✓ Tamaño máximo: 100 MB
✓ Detección de valores NaN excesivos (>50%)

Acciones:
• CSV válido → Se incluye en lista
• CSV inválido → Se salta con warning en logs
• Usuario informado a través de logs

MONITOREO (Opcional para aplicaciones 24/7)
═══════════════════════════════════════════════════════════════════════════════

Si quieres que la aplicación monitoree cambios en tiempo real:

from csv_auto_loader import watch_for_new_csvs

def on_change(evento, datasets):
    print(f"Cambio detectado: {evento}")
    print(f"Datasets ahora: {list(datasets.keys())}")

watch_for_new_csvs(callback=on_change, poll_interval=60)

Nota: Requiere instalar watchdog:
  $ pip install watchdog

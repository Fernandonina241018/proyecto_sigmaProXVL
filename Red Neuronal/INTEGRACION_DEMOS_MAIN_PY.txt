  CAMBIOS REALIZADOS
================================================================================

1. NUEVOS IMPORTS (Líneas ~21-28)
───────────────────────────────────────
   - interpretability (explain_predictions)
   - anomaly_detector (detect_outliers, data_quality_report, etc.)
   - Variable DEMOS_AVAILABLE (verifica si están instalados)
   - Variable global _LAST_MODEL

2. FUNCIONES WRAPPER (Líneas ~407-505)
───────────────────────────────────────
   - run_interpretability_demo(resultado)
     * Ejecuta SHAP + Partial Dependence
     * Usa splits['X_train'] del modelo actual
   
   - run_anomaly_demo(resultado)
     * 4 demos: Data Quality, Outliers, Class Imbalance, Prediction Anomalies
     * Usa splits del modelo actual
   
   - _update_last_model(resultado)
     * Actualiza variable global con el último modelo entrenado

3. MENÚ PRINCIPAL ACTUALIZADO (Líneas ~541-553)
───────────────────────────────────────
   Antes:
     Opciones: 1) Entrenar 2) Cargar 3) Ver historial 4) Eliminar 0) Salir
   
   Ahora:
     Opciones: 1) Entrenar 2) Cargar 3) Ver historial 4) Eliminar 
              5) Demo INTERPRETABILIDAD 6) Demo ANOMALY 0) Salir

4. LÓGICA DE OPCIONES 5 y 6 (Líneas ~763-785)
───────────────────────────────────────
   - Opción 5: Ejecuta run_interpretability_demo(_LAST_MODEL)
   - Opción 6: Ejecuta run_anomaly_demo(_LAST_MODEL)
   - Verifica que hay modelo entrenado antes de ejecutar

5. PREGUNTA POST-ENTRENAMIENTO (Líneas ~735-752)
───────────────────────────────────────
   Después de entrenar, pregunta automáticamente:
   
   ========================================================
     🎯 ¿QUIERES VER LAS DEMOS?
   ========================================================
     1) Ver interpretability (SHAP + Partial Dependence)
     2) Ver anomaly detector (outliers + quality)
     3) Volver al menú principal

6. ACTUALIZACIÓN DEL MODELO (Línea ~705)
───────────────────────────────────────
   - Después de entrenar, llama a _update_last_model(resultado)

================================================================================
  VERIFICACIÓN DE ERRORES
================================================================================

Si al ejecutar ves...
───────────────────────────────────────
"❌ Módulo interpretability no disponible"
  → Faltan instalar: pip install shap

"❌ Módulo anomaly_detector no disponible"
  → Faltan instalar los módulos: anomaly_detector.py, interpretability.py

"❌ No hay modelo entrenado"
  → No has entrenado. Ejecuta opción 1 primero.

"❌ No hay datos disponibles"
  → El split no está disponible. Entrena de nuevo.

"ℹ️  Demos no disponibles (módulos no instalados)"
  → Revisa que interpretability.py y anomaly_detector.py existan

================================================================================
  VARIABLES CLAVE DESPUÉS DE prepare_data()
================================================================================

Función:
    X_raw, y_processed, preprocessor, meta = prepare_data(
        data_with_target,
        target='aprobado',
        problem_type='auto'
    )

Variables retornadas:
    X_raw: DataFrame con features (sin target)
           - Tamaño: (n_samples, n_features)
           - Tipo: pandas.DataFrame
           - Ejemplo: X_raw.shape = (100, 3)

    y_processed: Array con target codificado
                - Tamaño: (n_samples,)
                - Tipo: numpy.ndarray
                - Valores: [0, 1] para binary

    preprocessor: ColumnTransformer ajustado
                 - Tipo: sklearn.compose.ColumnTransformer
                 - Función: procesar nuevos datos igual que training
                 - Uso: preprocessor.transform(X_new)

    meta: Dict con metadatos
          - problem_type: 'binary' | 'multiclass' | 'regression'
          - num_features: lista de features numéricas
          - cat_features: lista de features categóricas
          - target_classes: [0, 1] para binary
          - feature_baselines: valores base por feature

💡 EXPLICACIÓN TÉCNICA
================================================================================

El error "too many values to unpack (expected 3, got 4)" ocurre cuando:

1. Una función retorna N valores
2. Intentas desempaquarlos en M variables
3. N ≠ M

En Python:
    a, b, c = [1, 2, 3]        ✅ OK (3 valores, 3 variables)
    a, b, c = [1, 2, 3, 4]     ❌ Error (4 valores, 3 variables)
    a, b, c, d = [1, 2, 3, 4]  ✅ OK (4 valores, 4 variables)

En nuestro caso:
    ❌ preprocessor, X_proc, meta = prepare_data(...)
       Esperando 3 valores, pero prepare_data() retorna 4

    ✅ X_raw, y_train, preprocessor, meta = prepare_data(...)
       Esperando 4 valores, prepare_data() retorna 4

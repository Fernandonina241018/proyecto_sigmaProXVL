🐛 SOLUCIÓN DE PROBLEMAS
═══════════════════════════════════════════════════════════════════════════

Problema: ModuleNotFoundError: No module named 'model_manager'
Solución: Asegúrate de estar en la carpeta correcta (Red Neuronal/)

Problema: Carpeta modelos_guardados no aparece
Solución: Se crea automáticamente al entrenar el primer modelo

Problema: No puedo cargar el modelo
Solución: Verifica el ID exacto (ej: modelo_001)

Problema: Error de permisos
Solución: chmod -R 755 modelos_guardados/

Problema: JSON corrupto
Solución: Elimina modelo_registro.json y regenéralo

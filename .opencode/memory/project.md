# 🧠 Código de Memoria - Proyecto SigmaProXVL

## Reglas
- Actualizar automáticamente tras cada tarea exitosa
- Solo registrar hechos verificados y confirmados por el usuario
- Máximo 200 líneas
- Priorizar: estado del proyecto, decisiones del usuario, correcciones
- Incluir fechas y referencias a commits cuando sea posible
- Eliminar duplicados y mantener el más reciente
- Formato: secciones claras con viñetas
- No registrar credenciales, rutas de sistema operativo, o información sensible
- Incluir timestamps ISO
